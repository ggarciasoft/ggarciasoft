## 🚀 Featured Projects

### 🌐 [OrchestFlowAI](https://github.com/ggarciasoft/orchest-flow-ai)
> Open-source, self-hosted enterprise AI workflow platform

A modular platform for building AI-driven business workflows using reusable nodes. Features a visual drag-and-drop designer, an asynchronous workflow execution engine, 31 built-in nodes (AI, human approvals, integrations, documents, logic), full RBAC, and a provider-agnostic LLM abstraction supporting OpenAI, Anthropic, Azure, and Ollama.

`C#` `.NET 9` `Next.js` `React` `TypeScript` `PostgreSQL` `Redis`

---

### 🤖 [Agentic Release Governance Platform](https://github.com/ggarciasoft/agentic-release-governance-platform)
> MCP-first multi-agent release assistant for Azure DevOps

An agent-host-agnostic release system that helps teams create release items, collect work items and pull requests, validate readiness, and generate release documentation — all through the Model Context Protocol. Runs on GitHub Copilot, Cursor, and Claude.

`C#` `.NET 9` `MCP` `Azure DevOps` `Multi-agent`

---

### 📬 [MailLedger](https://github.com/ggarciasoft/mail-ledger-api)
> Turn email into structured financial intelligence

A SaaS application that connects to Gmail/Outlook, syncs messages, and uses OpenAI to classify financial emails and extract transaction details into an immutable ledger. Includes JWT authentication, Google/Microsoft SSO, Stripe subscriptions, real-time SignalR updates, and outbound webhooks.

`C#` `.NET 9` `React` `TypeScript` `PostgreSQL` `OpenAI` `Stripe`  
→ Frontend: [mail-ledger-ui](https://github.com/ggarciasoft/mail-ledger-ui)
